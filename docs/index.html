freeCashFlowTool

A lightweight Stock Analysis Dashboard that visualizes prices, free cash flow, FCF yield and profitability metrics.

Quickly visualize financial metrics with Plotly and Streamlit — ideal for rapid prototyping and research.

View Source on GitHub

Key Features

  • Interactive charts (line & bar) with Plotly
  • Free Cash Flow & FCF Yield analysis
  • Profitability margin tracking (Gross / Operating / Net margin)
  • Easy local startup & Docker support

Quick Start

Run locally with a virtual environment or run within Docker for guaranteed parity across machines.

python3 -m venv my_project_env
source my_project_env/bin/activate
pip install -r requirements.txt
pip install -r requirements-dev.txt
streamlit run app.py

Docker

Build & test in a containerized environment:

docker build -t freecashflow-tool .
docker run --rm -v "$(pwd):/app" freecashflow-tool

For Collaborators

We added a small scripts/ folder to create the local venv and run tests. Use ./scripts/bootstrap.sh to set up the environment and ./scripts/run_tests.sh to run the test suite.

Installation & Tips

Set environment variables to access external APIs. Example:

cp .env.example .env
# then edit .env to include your keys
