## What is freeCashFlowTool?

freeCashFlowTool is a small Streamlit dashboard that displays historical stock price data and financial metrics (e.g., Free Cash Flow, FCF Yield, Profitability Margins), using Plotly for interactive charts. The GitHub Pages landing site provides an overview and a quick way to access the live Streamlit app, or explore the source.
